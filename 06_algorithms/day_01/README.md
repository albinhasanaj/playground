# Day 01: Sorting Algorithms
Implement and analyze classic sorting algorithms.

## Algorithms
- Bubble Sort O(n²)
- Selection Sort O(n²)
- Insertion Sort O(n²)
- Merge Sort O(n log n)
- Quick Sort O(n log n) average
- Heap Sort O(n log n)

## Math Focus
- Time complexity analysis
- Space complexity
- Stability
