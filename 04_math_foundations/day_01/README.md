# Day 01: Matrix Operations

## Learning Objectives

- Represent matrices in C++ (row-major vs column-major)
- Implement basic matrix operations
- Understand matrix algebra fundamentals
- Handle edge cases and errors

## C++ Concepts

- 2D data storage strategies
- Template classes for generic matrices
- Operator overloading for natural syntax
- Exception handling for invalid operations

## Math Concepts

- **Matrix representation**: MxN grid of numbers
- **Addition/Subtraction**: Element-wise, same dimensions required
- **Multiplication**: (M×K) × (K×N) = (M×N)
- **Transpose**: Swap rows and columns
- **Identity matrix**: Multiplicative identity
- **Determinant**: Scalar value, invertibility
- **Trace**: Sum of diagonal elements

## Your Task

Implement a Matrix class:
1. Generic MxN matrix storage
2. Addition, subtraction (same size)
3. Matrix multiplication
4. Scalar operations
5. Transpose, determinant (for square matrices)
