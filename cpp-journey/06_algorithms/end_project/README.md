# Level 06 End Project: Algorithm Benchmark Suite

Complete algorithm benchmarking suite with Python visualization.

## Features
- All sorting algorithms benchmarked
- Searching algorithms compared
- Graph algorithm performance analysis
- CSV output for data analysis
- matplotlib charts for visualization
