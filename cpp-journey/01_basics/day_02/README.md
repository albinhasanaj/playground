# Day 02: Loops and Boolean Logic

## Learning Objectives

- Master `for`, `while`, and `do-while` loops
- Understand loop control: `break` and `continue`
- Work with boolean expressions and short-circuit evaluation
- Practice nested loops

## C++ Concepts

- **for loop**: `for (init; condition; increment) { body }`
- **while loop**: `while (condition) { body }`
- **do-while loop**: `do { body } while (condition);`
- **Boolean type**: `bool` with values `true` and `false`
- **Short-circuit evaluation**: `&&` stops on first false, `||` stops on first true

## Math Concepts

- **Summation**: Computing Σ(i) from 1 to n
- **Factorial**: n! = n × (n-1) × ... × 1
- **Prime checking**: Testing divisibility
- **Fibonacci sequence**: F(n) = F(n-1) + F(n-2)

## Your Task

Implement mathematical computations using loops:
1. Sum of first n natural numbers
2. Factorial of n
3. Check if a number is prime
4. Generate first n Fibonacci numbers
