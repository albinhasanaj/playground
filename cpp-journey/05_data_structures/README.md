# Level 05: Data Structures

Build custom data structures from scratch in C++ with TypeScript CLI integration.

## Days
- Day 1: Dynamic Arrays (vector implementation)
- Day 2: Linked Lists (singly and doubly)
- Day 3: Hash Maps (open addressing, chaining)
- Day 4: TypeScript CLI tool using C++ containers

## End Project
Full STL-style container library with Node.js bindings
