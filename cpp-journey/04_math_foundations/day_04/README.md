# Day 04: Python Visualization

## Overview
Use Python matplotlib to visualize matrix transformations and linear algebra concepts.

## Files
- `cpp/` - C++ matrix library with Python-callable exports
- `python/visualize.py` - matplotlib visualization scripts

## Tasks
1. Build C++ shared library with matrix exports
2. Call from Python to perform transformations
3. Visualize 2D transformations (translate, rotate, scale)
4. Plot eigenvalues and eigenvectors
