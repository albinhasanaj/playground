# Day 04: Functions

## Learning Objectives

- Define and call functions in C++
- Understand function parameters (by value)
- Work with return values and return types
- Learn function overloading
- Understand default parameters

## C++ Concepts

- **Function declaration**: `returnType functionName(parameterList);`
- **Function definition**: includes the body `{ ... }`
- **Overloading**: same name, different parameter types/counts
- **Default parameters**: `void foo(int x, int y = 10);`
- **Recursion**: function calling itself

## Math Concepts

- **Power function**: x^n computed iteratively and recursively
- **GCD**: Greatest Common Divisor using Euclidean algorithm
- **LCM**: Least Common Multiple via GCD
- **Combinatorics**: C(n, k) = n! / (k! × (n-k)!)
